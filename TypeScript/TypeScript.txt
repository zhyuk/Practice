# 타입스크립트
    : 자바스크립트 대용품. 자바스크립트 문법을 그대로 사용하며 타입부분을 다르게 사용하는 자바스크립트 업그레드 버전같은 느낌.
    
    자바스크립트는 Dynamic Typing이 가능하다.
        * Dynamic Typing : 5 - '3'을 계산할 때 숫자-숫자가 아니므로 불가능하지만, JS의 경우 알아서 숫자로 바꿔 계산해준다.
    
    코드가 길어지면 이런 자유도와 유연성은 오히려 좋지 않다. 

    타입스크립트 사용이유
        1. 타입스크립트는 엄격하기 때문에 에러메세지 퀄리티가 좋다.
            * var counter = 0; counter = '1'로 재선언한 경우, 숫자넣어야한다고  에러표출
            * append 대신 apend 작성 시 오타도 알려준다.
